Скачайте все файлы и не меняйте их расположение и названия.

Запуск локального сервера через Node.js:
1. Убедитесь, что у вас установлен Node.js.

2. Установите serv 
 Откройте cmd и впишите туда "npm install -g serve"

3. Перейдите в папку вашего проекта
   cd путь/к/проекту

4. Запустите локальный сервер
   serve -s

5. Откройте сайт в браузере
   "http://localhost:3000"
